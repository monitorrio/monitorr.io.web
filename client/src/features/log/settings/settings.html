  • Install Monitorr Nuget Package

  • Copy LogId


    Copy

  • Paste Copied LogId in Projet Web.config

    <elmah> <security allowRemoteAccess="false" /> <errorLog type="monitorr.io.elmah.ErrorLog, monitorr.io.elmah" LogId="{{ctrl.logId}}"/> </elmah>

    Your log dashboard will start updating as soon as you application will start generating errors.
